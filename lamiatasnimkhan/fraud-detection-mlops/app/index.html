Service Topology

3 services on the mlops-net bridge network, defined in docker-compose.yml

mlflow
Tracking server + artifact store. Receives runs from trainer, serves the registered FraudDetector model to api.
:5000 ghcr.io/mlflow/mlflow:v2.13.0
trainer
One-shot job: load CSV, preprocess, fit XGBoost, log everything to MLflow, register the best model. Exits 0 on success.
job fraud-trainer:latest
api
FastAPI service. Loads FraudDetector/Production from MLflow on startup, exposes /predict and /health.
:8000 fraud-api:latest
