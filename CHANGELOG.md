## How to Update This File

When creating a new release:

1. Update the version number in all relevant files
2. Add an entry for the new version with the current date
3. Move items from "Unreleased" to the new version section
4. Create a new "Unreleased" section

Example format for new releases:

```markdown
## [X.Y.Z] - YYYY-MM-DD

### Added
- Feature description

### Changed
- Change description

### Fixed
- Bug fix description
```
